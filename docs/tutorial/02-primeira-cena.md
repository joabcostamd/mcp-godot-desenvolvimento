# Tutorial 2: Sua primeira cena

**Tempo estimado:** 15 minutos
**Pré-requisito:** Tutorial 1 concluído (projeto aberto, MCP conectado)
**O que você vai ter no final:** Uma cena com personagem que você pode controlar.

---

## Passo 1: Criar uma cena vazia

No chat, diga:

```
criar cena nova
```

**O que vai acontecer:** O MCP cria uma cena 2D vazia e abre no Godot.

**Confira:** O editor do Godot mostra uma tela azul (o fundo padrão). No canto
superior esquerdo, o nome da cena aparece na aba.

---

## Passo 2: Adicionar um personagem

Agora peça um personagem:

```
criar um personagem controlavel — quadrado azul, mover com setas
```

**O que vai acontecer:** O MCP cria um personagem com:
- Um retângulo azul visível
- Controle pelas setas do teclado
- Colisão para não atravessar paredes

**Confira:** Você vê um quadrado azul no centro da tela do Godot.

---

## Passo 3: Rodar o jogo

Aperte **F5** no Godot (ou clique no botão **Rodar** no painel do MCP).

**O que vai acontecer:** O jogo abre em uma janela separada. Use as setas do
teclado para mover o quadrado azul.

**Confira:** O quadrado se move quando você aperta as setas. Se não mover,
volte ao chat e diga "o personagem não está se movendo".

---

## Passo 4: Ajustar o personagem

Agora personalize:

```
mude a cor do personagem para verde e deixe ele um pouco maior
```

**O que vai acontecer:** O MCP ajusta a cor e a escala do personagem.

**Confira:** Rode o jogo de novo (F5). O quadrado agora é verde e maior.

---

## Se algo der errado

- **O personagem não aparece:** Diga "o personagem não está visível na cena".
- **O personagem não se move:** Diga "o personagem não está respondendo ao teclado".
- **O jogo fecha sozinho:** Pode ser um erro no script. Diga "o jogo está fechando sozinho, verifique os erros".

---

## Pronto!

Você criou uma cena, colocou um personagem controlável, rodou o jogo e fez
ajustes. No próximo tutorial, você vai adicionar um inimigo.
