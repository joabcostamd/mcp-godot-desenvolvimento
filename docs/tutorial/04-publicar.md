# Tutorial 4: Publicar seu jogo

**Tempo estimado:** 10 minutos
**Pré-requisito:** Tutorial 3 concluído (jogo com personagem e inimigo)
**O que você vai ter no final:** Um arquivo executável do seu jogo, pronto para compartilhar.

---

## Passo 1: Verificar se está tudo pronto

No chat, diga:

```
release checklist
```

**O que vai acontecer:** O MCP verifica se falta algo importante:
- Se a cena principal está definida
- Se não há erros de compilação
- Se todos os recursos (imagens, sons) existem

**Confira:** O MCP responde com uma lista verde (tudo certo) ou amarela/vermelha
(coisas a corrigir).

---

## Passo 2: Exportar o jogo

Quando estiver tudo certo:

```
exportar o jogo para Windows
```

**O que vai acontecer:** O MCP prepara todos os arquivos e gera um executável
(.exe) na pasta do projeto.

**Confira:** No final, o MCP mostra o caminho do arquivo gerado, por exemplo:
`C:\MeuJogo\export\MeuJogo.exe`

---

## Passo 3: Abrir o jogo exportado

Vá até a pasta que o MCP mostrou e clique duas vezes no arquivo `.exe`.

**O que vai acontecer:** Seu jogo abre como um programa normal do Windows —
sem precisar do Godot, sem precisar do MCP. É um jogo de verdade.

**Confira:** O jogo abre e funciona igual a quando você apertava F5 no Godot.

---

## Passo 4: Compartilhar

Agora você tem um jogo pronto. Você pode:

- Mandar o arquivo para amigos testarem
- Publicar no itch.io (um site gratuito onde qualquer pessoa pode publicar jogos)
- Continuar melhorando — o MCP lembra de tudo que você já fez

Para continuar, é só voltar ao chat e pedir:

```
adicione mais 2 fases ao jogo
```

**O que vai acontecer:** O MCP retoma de onde parou e continua construindo.

---

## Se algo der errado

- **"Erro de compilação":** Diga "corrija os erros de compilação".
- **O export não gera o arquivo:** Verifique se os templates de exportação
  estão instalados. Diga "instale os templates de exportação do Godot".
- **O executável não abre:** Pode ser que seu antivírus tenha bloqueado.
  Tente desativar temporariamente ou adicionar uma exceção.

---

## Pronto!

Você instalou, criou um projeto, fez uma cena com personagem, adicionou um
inimigo, e publicou seu jogo. **Você terminou um jogo.** Isso é mais do que
a maioria das pessoas consegue.

Parabéns! 🎮
