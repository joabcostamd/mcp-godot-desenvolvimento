# As 6 fases do desenvolvimento

O MCP divide a criação do jogo em 6 fases. Isso existe para
proteger você: cada fase tem travas que impedem pular etapas
e garantem que nada importante seja esquecido.

## 1. IDEIA
Você descreve o jogo. O MCP entende o que você quer e monta o plano.

## 2. DESIGN
O MCP define as regras, personagens, fases e a estrutura do jogo.

## 3. PROTOTIPO
O MCP constrói uma versão jogável simples para testar a ideia.

## 4. CONTEUDO
O MCP adiciona fases, inimigos, itens, diálogos — o jogo cresce.

## 5. POLIMENTO
O MCP ajusta sons, efeitos visuais, menus e a experiência final.

## 6. PRONTO_PARA_LANCAR
O jogo está pronto. O MCP prepara a exportação.
