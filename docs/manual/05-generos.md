# Gêneros de jogo disponíveis

Escolha um gênero para começar, ou peça algo novo —
o MCP adapta o que você quiser.

**3 gêneros:**

## Plataforma 2D
Jogo de plataforma 2D com scroll lateral. O jogador corre, pula e enfrenta obstáculos e inimigos para chegar ao fim da fase. Gênero clássico: acessível para iniciantes, profundo para veteranos.

**Sistemas inclusos:**
- health
- player_controller

**Exemplo:** "quero fazer um jogo de plataforma 2d"

## Puzzle 2D
Jogo de raciocínio 2D com regras simples e profundidade crescente. O jogador resolve desafios lógicos ou espaciais para avançar. Ideal para mobile e web.

**Sistemas inclusos:**
- grid_manager

**Exemplo:** "quero fazer um jogo de puzzle 2d"

## Tiro Top-Down
Jogo de tiro com visão superior. O jogador se move livremente pelo mapa, mira e atira em inimigos que vêm de todas as direções. Gênero versátil: de bullet hell a Survivors-like.

**Sistemas inclusos:**
- health
- player_topdown

**Exemplo:** "quero fazer um jogo de tiro top-down"
