# Teleport

Teletransporte via `Area2D`. `body_entered` move o player para `target_position` ou troca de cena via `target_scene` + `SceneTransition`. Filtra por `player_group`.

**Parâmetros:** `target_position`, `target_scene`, `player_group`.

**Uso:** Posicione na cena. `target_scene` opcional para trocar de fase. Sinal `teleported`.

**Fontes:** Godot 4.7 ClassDB (`Area2D`, `SceneTransition`).
