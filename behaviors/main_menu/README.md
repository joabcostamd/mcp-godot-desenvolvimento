# Main Menu

Menu principal com botões Play, Settings e Quit em `CanvasLayer`. Configurável: `play_scene`, `settings_scene`, `show_quit`, `show_settings`. Usa `SceneTransition` para navegação entre cenas.

**Parâmetros:** `play_scene`, `settings_scene`, `show_quit`, `show_settings`.

**Uso:** Adicione à cena de título. Sinais `play_pressed`, `settings_pressed`, `quit_pressed`.

**Fontes:** Godot 4.7 ClassDB (`Button`, `VBoxContainer`, `CanvasLayer`).
