## What is it

**MCP Godot Agent** is a Python MCP server that acts as the **process owner** for developing an entire Godot game. It is not just a bridge between AI and engine — it has real gates that prevent skipping steps, automated verification, and step-by-step guidance.

**Who is it for:** People who want to create a game but do not know how to code. You describe what you want in natural language and the AI builds it.

**What it does:**
- Creates complete Godot projects from a single phrase
- Manages 6 development phases (Idea → Design → Prototype → Content → Polish → Ready)
- Offers 32 ready-to-use game genres
- Verifies each step automatically (compilation, tests, smoke test)
- Works offline for operations that do not depend on AI

## ⚡ Get started in 1 minute

```powershell
# 1. Install (only Python 3.11+ required, no dependencies)
python init.py

# 2. Open VS Code in the project folder

# 3. In Copilot chat, type:
/plan
```

**Available commands after installation:**

| Command | What it does |
|---|---|
| `/plan` | Plans the next step of your game |
| `/act` | Executes the approved plan |
| `/handoff` | Saves session progress |
| `/manual` | Full command reference |

**Alternative without installer:**
```powershell
git clone https://github.com/joabcostamd/mcp-godot-desenvolvimento
cd mcp-godot-desenvolvimento
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
.venv\Scripts\python server.py --profile dev
```
